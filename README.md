# Ayachi

Visual novels bot for the french discord community.

## tech stack

discord.js | mongoose

## setup

```bash
git clone https://github.com/L3ne/Ayachi.git
npm install
node index.js
```

## features

- xp system
- giveaway system
- tempvoice system
- bump system
- logs system
- steam discount notifier
- visual novels commands
- moderation commands
- MongoDB Database
