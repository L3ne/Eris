# Eris

Visual novels bot for the french discord community.

## tech stack

discord.js | mongoose

## setup

```bash
git clone this repo
npm install
node index.js / .
```

## features

- xp system
- visual novels commands
- moderation commands
- logs system
- MongoDB Database
